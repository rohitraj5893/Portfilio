# 🚀 Portfolio Website

A modern, responsive portfolio website built with React and Vite. Showcase your skills, projects, and services with this sleek and professional portfolio template.

## ✨ Features

- **Responsive Design** - Mobile-first approach ensuring great experience on all devices
- **Fast Performance** - Built with Vite for lightning-fast development and production builds
- **Tailwind CSS** - Utility-first CSS framework for rapid UI development
- **Icon Library** - React Icons for beautiful, scalable icons
- **Multiple Sections**:
  - Home/Hero Section
  - About Me
  - Tech Skills
  - Projects Portfolio
  - Services
  - Contact Section
  - Navigation & Footer

## 🛠️ Tech Stack

- **React** ^19.1.1 - UI library for building interactive components
- **Vite** ^7.1.7 - Modern build tool for fast development
- **Tailwind CSS** ^4.2.1 - Utility-first CSS framework
- **React Icons** ^5.6.0 - Icon library with thousands of icons
- **React DOM** ^19.1.1 - React package for working with the DOM

## 📋 Prerequisites

Before you begin, ensure you have the following installed:
- Node.js (v14 or higher)
- npm or yarn package manager

## 🚀 Getting Started

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd Portfolio
```

2. Install dependencies:
```bash
npm install
```

### Development

Start the development server:
```bash
npm run dev
```

The application will open at `http://localhost:5173` with hot module replacement (HMR) enabled.

### Build

Create a production build:
```bash
npm run build
```

### Preview

Preview the production build locally:
```bash
npm run preview
```

### Linting

Check code quality with ESLint:
```bash
npm run lint
```

## 🎨 Components Overview

- **Navbar** - Navigation menu with links to different sections
- **Home** - Hero section with introduction and call-to-action
- **About** - Personal background and professional summary
- **TeckSkill** - Display of technical skills and proficiencies
- **Project** - Portfolio of completed projects with descriptions
- **Services** - Services offered or expertise areas
- **Contact** - Contact form for inquiries
- **Footer** - Footer with links and information

## 🔧 Customization

### Update Project Data
Edit [src/data/ProjectData.js](src/data/ProjectData.js) to update your projects and portfolio information.

### Styling
The project uses Tailwind CSS. Modify [src/index.css](src/index.css) for global styles and adjust component classes as needed.

### Content
Update component files in `src/componets/` to customize content for each section.

## 📦 Deployment

### Build for Production
```bash
npm run build
```

### Deploy to Vercel (Recommended)
```bash
npm install -g vercel
vercel
```

### Deploy to Other Platforms
The `dist` folder generated by `npm run build` contains the optimized production build ready for deployment to any static hosting service.
